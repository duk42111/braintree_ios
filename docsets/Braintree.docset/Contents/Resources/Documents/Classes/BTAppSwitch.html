@interface BTAppSwitch : NSObject

@class BTAppSwitch @brief Handles return URLs when returning from app switch and routes the return URL to the correct app switch handler class. @discussion returnURLScheme must contain your app’s registered URL Type that starts with the app’s bundle ID. When your app returns from app switch, the app delegate should call handleOpenURL:sourceApplication:
